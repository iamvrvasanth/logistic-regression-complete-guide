# Logistic Regression - End-to-End Machine Learning Implementation

## Overview

This repository demonstrates a complete implementation of Logistic Regression using Scikit-learn. It covers model training, evaluation, hyperparameter optimization, multiclass classification, imbalanced datasets, and ROC-AUC analysis.

The project is designed to provide practical experience with one of the most widely used classification algorithms in machine learning.
